# 本章导读

本章会讲解 力扣/LeetCode 上大部分的二叉树习题，全部使用前文 [二叉树系列算法核心纲领](</zh/algo/essential-technique/binary-tree-summary/>) 中总结的「遍历」和「分解问题」的思维模式，帮助读者通过大量的习题来巩固这些思维方法，以二叉树为引子，深入理解递归算法，为后面章节介绍的其他算法做铺垫。

对于时间充裕的读者，我非常建议刷一下本章的习题，最好能动手做一做。不要害怕题多，只要你熟练使用正确的思维模式，这些习题都可以直接秒杀，非常容易建立信心。

对于时间比较紧迫的读者，只要理解了前面的二叉树相关文章，可以先跳过本章习题往后学习。不过未来有时间的话，还是建议回来看一看二叉树的习题。

**相信我，只要把二叉树结构玩明白，会让你的算法学习之路事半功倍** 。
