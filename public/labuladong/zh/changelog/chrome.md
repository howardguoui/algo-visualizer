# Chrome 刷题插件更新日志

Info

本页面为 Chrome 插件的更新日志，插件的详细使用指南见[Chrome 插件说明书](</zh/algo/intro/chrome/>)。

## ¶v6.0.2

  * 使用 iframe 渲染思路弹窗，优化用户体验，提供和网站相同的使用体验，网站会员不再需要额外的操作解锁插件。
  * 修复「禁用第三方 cookie」导致网站会员无法查看思路的问题。


## ¶v5.0.9

  * 适配新版力扣的题目列表。
  * 修复若干 bug。


## ¶v5.0.6

  * 修复部分题目不显示思路按钮的 bug。
  * 优化算法可视化面板缓存出错的问题。


## ¶v5.0.5

  * 修复 dev 版本的 Chrome 浏览器无法使用插件的 bug。
  * 提升海外用户拉取网站数据的速度。


## ¶v5.0.3

  * 修复部分题目详情页中，思路/题解按钮显示错位的问题。


## ¶v5.0.2

  * 修复 dev 版本的 Chrome 浏览器无法使用插件的 bug。


## ¶v5.0.1

  * 修复收藏题目列表无法渲染思路/题解按钮的 bug。


## ¶v4.4.7

重大提升

大幅提升思路中多语言解法的准确性。

## ¶v4.4.6

bug 修复

修复了插件思路中，文章链接还引用旧版课程链接的问题。按照 [Chrome 插件说明书](</zh/algo/intro/chrome/>) 中的方法，重新拉取数据即可。

## ¶v4.4.5

  * 优化数据拉取速度，优化若干用户体验。
  * 修复 studyplan 插件无法生效的 bug。


## ¶v4.4.0

重大更新

可视化面板新增「编辑」按钮，支持修改我的解法代码后直接在面板上运行，方便验证你的奇思妙想：

![](/images/algo/changelog/chrome-viz-editor.jpg)

Breaking Change

插件适配了我的新网站 [labuladong.online](<https://labuladong.online/>)，将不再支持小鹅通平台。

之前购买了小鹅通的读者，可以迁移课程权限到新网站，迁移指南 [见这里](<https://labuladong.online/algo/user/orders/>)。基于新网站解锁插件中课程专属题解的方法，已经在课程的第一章进行了更新。

## ¶v4.3.7

  * 修复 study plan 页面的 bug。
  * 修复调整灵动布局后就无法显示思路题解按钮的 bug。


## ¶v4.3.6

  * 优化用户体验，自动判断网速快的数据源，并在安装后自动拉取数据。
  * 适配新 UI 灵动布局（Dynamic layout）。
  * 优化插件弹窗的显示状态，修复了之前版本显示错误的问题： ![](/images/algo/changelog/chrome_popup.jpg)


## ¶v4.3.5

  * 修复在网页写代码时卡顿的 bug。
  * 支持 LeetCode/力扣 study plan 页面的渲染。


## ¶v4.3.2

  * Chrome 插件可以连接数据结构/二叉树课程，在网页上学习课程。


## ¶v4.3.0

  * 修复 Java 代码渲染颜色错误的 bug。


## ¶v4.2.0

  * 修复 LeetCode 和力扣的新 UI 导致插件失效的问题。

  * 思路弹窗中能够保存用户的编程语言偏好，下次会默认打开对应语言的解法代码，不必频繁切换。

  * 支持 400 道题的算法可视化，持续增加中： ![](/images/algo/visualize_intro/chrome_example.jpg)


## ¶v4.1.0

  * 修复剑指 offer 题目的思路解法无法显示的 bug。
  * 支持所有主流编程语言的解法代码。 ![](/images/algo/changelog/chrome_multi_lang.jpg)
