## Recommended Batch Size

**10–12 problems per session.**

Reasoning:
- Each problem = ~60–80 lines of TypeScript (description, examples, constraints, JS starter, Python starter, test cases, hint)
- 10–12 problems = ~700–900 lines added per session — fits comfortably without exhausting tokens
- Leaves room for TypeScript check + commit + push in the same session

**Priority order:** Hot 100 → Quick Master → Beginner
